# About

## DiversityLens

DiversityLens was developed as part of a Master's thesis at Fachhochschule Dortmund in collaboration with the DFKI Robotics Innovation Center in Bremen.

## Purpose

The tool is designed exclusively as a dataset auditing tool for improving fairness in AI systems. It is not intended for individual identification, surveillance, or any form of profiling.

## Technology Stack

- **DeepFace** — Face detection and demographic estimation
- **RetinaFace** — Face detection backend (selected for accuracy over speed)
- **Bokeh** — Interactive visualization dashboard
- **pandas** — Data processing and age group binning
- **OpenCV** — Video frame extraction
